# Бандл PostBundle

пример поста
{
    "id":null,
    "title":"Тест ",
    "tags":"[{\"name\":\"1\"},
        {\"name\":\"2\"}]",
    "attachments":"[{"type\":"attachment-text","value":"111111"}]"}

отвечает за создание поста состоящего из
title - заголовка
tags - [] тегов
attachments - [] аттачменты прикрепления
